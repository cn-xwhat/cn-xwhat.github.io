invoke

invoke和getMethod一样，第二个参数可加可不加，具体看情况来定（是否为无参方法），而第一个参数一般都为**clazz.newInstance()**，

它的作用就是调用这个类的无参构造函数。所以第一个参数为类的对象，第二个参数为方法的参数（需要的话）

但实际中**clazz.newInstance()**经常不成功，原因如下

  1. 你使用的类没有无参构造函数
  2. 你使用的类构造函数是私有的

getConstructor

在一个类没有无参构造方法，也没有类似单例模式里的静态方法的情况下，即有 有参构造方法，并且是public。可以使用getConstructor来实例化对象。

  • getMethod 系列方法获取的是当前类中所有公共方法，包括从父类继承的方法
  • getConstructor获取当前类的给定公共构造方法
  • getDeclaredMethod 系列方法获取的是当前类中“声明”的方法，是实在写在这个类里的，包括私 有的方法，但从父类里继承来的就不包含了（下面会用到）
